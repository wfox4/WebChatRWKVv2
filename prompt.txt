```
open System.Text.RegularExpressions

let isValidEmail (email: string) =
    let emailRegex = new Regex("^([a-zA-Z0-9_\\-\\.]+)@([a-zA-Z0-9_\\-\\.]+)\\.([a-zA-Z]{2,5})$")

    match emailRegex.Match(email) with
    | matchResult when matchResult.Success -> true
    | _ -> false
```

The regular expression used in this program checks if the email has the following pattern:

```
^([a-zA-Z0-9_\\-\\.]+)@([a-zA-Z0-9_\\-\\.]+)\\.([a-zA-Z]{2,5})$
```

This pattern matches an email address that starts with one or more alphanumeric characters, underscores, hyphens, or dots, followed by an at symbol (@), then another string of one or more alphanumeric characters, underscores, hyphens, or dots, followed by a period (.), and finally a string of two to five alphabetic characters.

To use this program, you can call the `isValidEmail` function with a string parameter that contains the email you want to check, like this:

```

```

```
password = input("Enter your password: ")
```

- The password contains at least one lowercase letter
- The password contains at least one special character
